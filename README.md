# Anime Project

## Description

Anime scheduling, information and trailers

## Dependencies

[Node]("https://nodejs.org/en/")

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```
## Built with

**Client:** React, GraphQL

**API** - [AniList]((https://anilist.gitbook.io/anilist-apiv2-docs/overview/graphql/getting-started))
